Python学习之配置Mysql

发表于 2017-12-13 |

之前开发还是JDBC用的多一些，这边Python-mysql也是大同小异，只会更简便。

遇到了一些问题记录一下，也是复习，参考博客：python使用mysql数据库

MySql环境配置

首先直接pip行不通——可能是我pip版本问题？显示找不到对应版本的MySQLdb。

那就自己从官网下载MySQL-python 1.2.x，随即终端中运行python setup.py install即可。但这边遇到了一些问题：

  • 找不到mysql_config.解决方式:ln -s /usr/local/mysql/bin/mysql_config /usr/local/bin/mysql_config把mysql_config的路径链接一下

  • 找不到setup_posix.py（准确的说是指令被拒绝）.解决方式：修改该文件的权限为可执行chmod a+x ./setup_posix.py

之后就可以正常去操作了。
