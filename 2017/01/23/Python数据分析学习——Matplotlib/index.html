Python数据分析学习——Matplotlib

发表于 2017-01-23 |

进入Python数据分析学习的第二部分——Matplotlib的学习。它是非常强大的Python画图工具。
以下是此次的学习笔记。

基本使用

基本用法

关键代码：

其中：使用plt.figure定义一个图像窗口；使用plt.plot画(x ,y)曲线；使用plt.show显示图像。

figure图像

以plt.figure()打头，后面的内容都是与这个figure相关的。
plot有很多参数，如——num（figure序号）、figsize=(长，宽)、color、linewidth以及linestyle（’–’是虚线）等等。

设置坐标轴

坐标轴的范围——plt.xlim／plt.ylim。

坐标轴名称——plt.xlabel／plt.ylabel。

刻度——plt.xticks／plt.yticks。可以传入文字，与数字一一对应。在前后加上$可以转变为机器可以读的。空格前加上\，再在前面加r表面是正则表达式。阿尔法——\alpha。

获取当前坐标轴信息——plt.gca（get current axis ）

设置边框——ax.spines[]。再.set_color设置颜色，其中[]可以取top、bottom、left以及right分别代表上下左右。

设置x(y)坐标刻度数字或名称的位置——ax.x（y)axis.set_ticks_position。所有位置：top，bottom，both，default，none。

ax.set_position设置边框位置。位置所有属性：outward，axes（半分比），data（值）。

Legend图例

为了展示每个数据对应的图像名称。

先对每条线指定label参数，随后用plt.legend()即可在图上添加图例。该函数的其它参数：handles——、labels、loc——位置（upper right等等十种／best会找一个数据不密集之处放置）。

分别重新设置线条对应的label：首先在两条线的plot代码中用变量存下来，且要以逗号结尾（ 因为plt.plot() 返回的是一个列表）。

Annotation标注

Annotation
plt.annotate。参数：内容、xy——待标注点的位置、xycoords——基于什么来选位置、xytext——标注内容相对于标注点偏差值、textcoords——对于标注位置的描述、fontsize、arrowprops——设置箭头的相关参数。

Text
plt.text。参数：前两个是添加text的位置、内容、fontdict设置文本字体。

Tick能见度

设置bbox参数来调节图像信息，使得图片更加易于观察。

首先获取坐标轴刻度的label。可以设置其字体大小——label.set_fontsize()、透明度相关——label.set_bbox()（facecolor调节前景色／edgecolor设置边框／alpha设置透明度）。

图画种类

散点图

使用plt.scatter()。参数：x和y作为点位置的坐标、s、c、cmap、alpha等等。

补充：

生成标准正态分布二维数据组的方法——np.random.normal(0,1,n)／均匀分布是np.random.uniform(...)

隐藏x刻度：plt.xticks(()),y同理。

柱状图

使用plt.bar()。参数：x和y作为点位置的坐标、facecolor、edgecolor。

在柱体上方（下方）加上数值——plt.text。横向对齐ha；纵向对齐va。

注：for x,y in zip(X,Y)会把X,Y的数据值分别传入x和y中

等高线图

颜色填充——contourf。参数：x和y作为点位置的坐标、高度、等高线的密集程度、透明度、cmap。

等高线绘制——contour。参数：x和y作为点位置的坐标、高度、等高线的密集程度、colors、linewidth。

数值标签——clabel。参数：等高线、inline——是否画在线里面、字体。

注：meshgrid用来在二维平面中将每一个x和每一个y分别对应起来，编织成栅格。

图片

绘制图片——imshow。参数：2D-array、interpolation、cmap、origin——可以取lower或者upper代表的就是选择的原点的位置

图片注释——colorbar。参数：shrink——长度百分比。

3D数据

导入Axes 3D模块，且同样需要给X和Y值，并将X和Y编织成栅格。

绘制3D图像——ax.plot_surface。各种参数：X、Y、Z、rstride——行跨度，列同、cstride、cmap（彩虹色——plt.get_cmap('rainbow')）。

还可以添加等高线，zdir设置从哪个轴投下。

多图合并显示

Subplot多合一显示

创建子图——plt.subplot()。参数：行、列、当前位置。

希望小图大小不一样，比如原来是两行两列放四张，而现在希望第一张占一行，余下三张占一行——先使用plt.subplot(2,1,1)将图像分成两行一列，当前位置1；再使用plt.subplot(2,3,4)将整个图像窗口分为2行3列, 当前位置为4(注意，此处而不是2)。

Subplot分格显示

subplot2grid:
使用plt.subplot2grid。参数：行列、起始点（索引值）、colspan、rowspan。

gridspec:
首先导入matplotlib.gridspec,并简写成gridspec。再用gridspec.GridSpec()将图像划分为几行几列，最后用plt.subplot来作图, 举例：gs[0, :]表示这个图占第0行和所有列,有点类似于切片。

通过设置左下角的位置以及宽高占整个figure坐标系的百分比——ax1 = fig.add_axes([left, bottom, width, height])或者plt.axes([...])来实现。

次坐标轴

主次坐标轴——同个图上有第2个y轴存在。

对默认坐标系使用twinx生成镜面效果后的ax，再接着将两幅图分别画在这两个ax上。
