从今天起，将配合莫烦的视频学习Python用以数据处理及显示的三个包——Numpy & Pandas、Matplotlib。
在这个过程中也会继续巩固之前学习的Python基础语法，并且重点是实践。
以下是第一部分Numpy & Pandas的学习。

Numpy & Pandas简介

为什么要使用这两个模块

  • 运算速度快——用C语言写的。

  • 消耗资源少——采用矩阵的计算，比自带的快很多。

安装

Mac上：sudo pip install numpy以及sudo pip install pandas。

Numpy学习

属性

numpy的几个属性：

ndim——维度。

shape——行数和列数。

size——元素个数。

使用numpy前需要导入模块——import numpy as np（为了方便采用np简写）。

使用np.array(...)可以将列表转为矩阵。

创建array

几个关键词：

array——创建数组。像创建列表一样，区别在于打印出来的每逗号。

dtype——指定数据类型。可以是int、float等等。

zeros——创建数据全为0。指定size时一定要用括号括起来。

ones——创建数据全为1。

empty——创建数据接近0。

arange——按指定范围创建数据。从哪到哪（不包括）以及步长，不需要额外的括号。

reshape——改变数据的形状。

linspace——创建线段。从哪到哪（包括）以及分成的段数。

基础运算

一维的情况：

加减乘除同往常一样。平方（几次方）用**。

numpy自带的sin、cos等函数也很方便使用。

print进行逻辑运算输出布尔矩阵，判断是否相等时要用==。

二维的情况：

其它基本类似，除了乘法有两种情况——逐个元素相乘a*b／矩阵的乘法np.dot(a,b)或者a.dot(b)。

np.random.random((a,b))——生成a行b列元素大小为0-1的矩阵。

numpy自带sum、min、max可用于求出矩阵中所有元素的和、最大以及最小值。如果需要单独求某行或者某列，需要指定维数axis，等于1代表每一行中，等于0表示每一列中。

索引——argmin、argmax。

平均值——mean/average。也可以指定axis。

中位数——median。

累计和——cumsum。

累差——diff。

找出非零的数——nonzero。将所有非零元素的行与列坐标分割开，重构成两个分别关于行和列的矩阵。

排序——sort。按行排序。

转置——np.transpose/a.T。

截——clip（矩阵，最小，最大）所有小于最小的都等于最小；所有大于最大的都等于最大。

索引

输出具体元素：

类似Python数组和列表的索引方式。

切片输出：

使用for row in a:遍历每一行；使用for column in a.T遍历每一列；如果要便利每一行，不能直接用for item in a:因为它还是一行一行输出。而要用for item in a.flat:——A.flatten()/A.flat前者返回array类型，后者是一个迭代器（object属性）。作用都是将一个矩阵变成数列。

合并array

上下合并——vstack。要求tuple，需要小括号括起。

左右合并——hstack。

不能用transpose把一个序列变成一个矩阵。

newaxis——加维度。［:,np.newaxis］在列上加一个维度／［np.newaxis,;］在行上加一个维度。可以直接加在array()后面

多个array纵向或横向的合并——concatenate。可以要求在某一个维度合并。axis=0是上下。

分割array

等分分割——split。三个参数（array，分成片段个数，横向还是纵向）axis＝1是纵向。

不等分分割——array_split。

横向分割——vsplit。

纵向分割——hsplit。

Numpy copy & deep copy

=赋值会有关联性。一个变了其它也会变。一旦赋值两者就是同一个变量，可用b is a判断。

copy——深度copy。没有关联性。

Pandas学习

基础知识

与Numpy的区别

Numpy是已经序列化好的矩阵或者序列。Pandas像是字典形式的Numpy。可以给其不同的行列命名。

Series的创建

pd.Series注意要大写。输出时会有序号，最下面会有类型dtype。(Numpy中是以float64/32保存)

DataFrame的创建

date_range函数——可以指定开始时间（如：’2017-01-20’）以及持续天数。

DataFrame是一个大型的矩阵，有行索引和列索引，均可以用index和columns来定义。如果为定义则默认是从零开始的整数。还可以用列表的形式来定义。

DataFrame相关属性：类型——dtypes;输出所有行名——index;输出所有列名——columns;所有的值——values;功能：描述——describe()。只能运算数字形式。

DataFrame也同样能transpose。

按索引排序——sort_index(axis=?,ascending=？)。可以指定升序或者降序以及按行还是按列排序。

按值排序——sort_values(by=’?’)。可以指定对哪一列进行排序。

选择数据

选择列——举例df['A'] 或者 df.A。

可以切片，指定行数或index区间即可。

根据标签进行选择——loc。横向纵向都可以。也可以同时指定两者。

根据位置进行选择——iloc。切片也可以，逗号分隔可以一个一个取出不连续的数据。

将标签和位置综合起来选择——ix。

是非选择——df[布尔表达式]

设置值

依据上一节内容选择到数据进行更改即可。

增加一列只需要赋值即可实现，但是长度必须对齐。

处理丢失数据

直接去掉有NaN的行或者列——dropna。还可以指定how,any或者all。

填充丢失数据——fillna。可以通过指定value来指定缺失处填充数据的值。

检查是否有缺失数据——isnull。会返回一个矩阵同等大小的布尔矩阵，缺失处为True。还可以进一步用np.any()=＝True来判断里面是否存在缺失值。

导入导出

pandas可以读取与存取很多种资料格式如csv、excel、json、html与pickle等。具体见官方说明文件

读入——read_csv(…)。会自动加索引。把第一行作为列名。

写出——to_pickle(…)。

合并concat

使用concat。其中有各种参数供调整：

横纵——axis。

索引重复解决——ignore_index=True，忽略原有索引，重新排序。

合并方式——join。有inner以及outer两个选项。默认是outer，没有column的用NaN填充inner只寻找有相同column的合并。

依照axes合并——join_axes。

添加数据——append。只有在纵向追加，也可以合并series。

合并merge

使用merge。类似于数据库的连接。

针对两组有key column的数据进行合并。on=...

一组key时，key位于中间，两边分别是对应的两组数据（除key以外）。

两组key时，有四种连接方式。how = ['left', 'right', 'outer', 'inner']，默认how=’inner’。

显示如何进行merge——indicator=True 或者indicator='...' 指定该列的名称。

根据index进行merge——left_index=True, right_index=True。

解决overlapping——使用suffixes，为相同的列名后加区分。

plot出图

首先import matplotlib.pyplot as plt，对普通的Series以及DataFrame只需要data.plot()再plt.show()一下即可。

打印出前几个数据——data.head()。默认是前五个。可另外指定。

plot的各种方法——bar、hist、box、kde、area、scatter、hexbin。其中scatter一般只有x和y两个属性。 关于其它方法见手册
