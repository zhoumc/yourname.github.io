前端学习笔记（四）

发表于 2018-03-06 |

应该是前端学习的最后一篇。重点学习一下bootstrap框架。

插一句，前端代码规整可以下一个插件HTMLBeautify快捷键是command+shift+h。

bootstrap

bootstrap 简介

Bootstrap是由Twitter的 Mark Otto 和 Jacob Thornton 开发的。Bootstrap 是 2011 年八月在 GitHub 上发布的开源产品。

bootstrap 安装

让一个常规的 HTML 文件变为使用了 Bootstrap 的模板：

bootstrap CSS

概述

为了让 Bootstrap 开发的网站对移动设备友好，确保适当的绘制和触屏缩放，需要在网页的 head 之中添加viewport meta标签:

<meta name="viewport" content="width=device-width, initial-scale=1.0">

width 属性控制设备的宽度。那么将它设置为 device-width 可以确保它能正确呈现在不同设备上。

initial-scale=1.0 确保网页加载时，以 1:1 的比例呈现，不会有任何的缩放。

通过添加 img-responsive class 可以让 Bootstrap 3 中的图像对响应式布局的支持更友好：

<img src="..." class="img-responsive" alt="响应式图像">

网格系统

Bootstrap 提供了一套响应式、移动设备优先的流式网格系统，随着屏幕或视口（viewport）尺寸的增加，系统会自动分为最多12列。

工作原理：通过一系列包含内容的行和列来创建页面布局。

  • 行必须放置在 .container class 内，以便获得适当的对齐（alignment）和内边距（padding）

  • 使用行来创建列的水平组

  • 内容应该放置在列内，且唯有列可以是行的直接子元素。

  • 预定义的网格类，比如 .row 和 .col-xs-4，可用于快速创建网格布局。

基本结构：

其中col-*-*:中间可取xs、sm、md、lg分别代表超小设备手机、小型设备平板、中型台式电脑和大型…..
后面可取1-12代表这列所占空间。

排版

内联子标题——在元素两旁添加<small>，或者添加 .small class，这样子您就能得到一个字号更小的颜色更浅的文本

强调文本——添加class="lead"

另外，强调标签<small>（设置文本为父文本大小的 85%）、<strong>（设置文本为更粗的文本）、<em>（设置文本为斜体）。

缩写——<abbr>在文本底部的一条虚线边框，当鼠标悬停在上面时会显示完整的文本。

引用——<blockquote>。其他选项包括，添加一个<small>标签来标识引用的来源，使用class .pull-right向右对齐引用。

列表：使用class .list-unstyled来移除样式。

代码

Bootstrap 允许您以两种方式显示代码：

  • 第一种是<code>标签。内联显示代码。

  • 第二种是<pre>标签。需要被显示为一个独立的块元素或者代码有多行。

表格

table类——最简单的

.table-striped类——可以在<tbody>行上看到条纹

.table-bordered类——看到每个元素周围都有边框

.table-hover类——当指针悬停在行上时会出现浅灰色背景

.table-condensed类——行内边距（padding）被切为两半，让表看起来更紧凑

.table-responsive类——可以让表格水平滚动以适应小型设备

表单

1、垂直表单：

向父<form>元素添加 role=”form”。

把标签和控件放在一个带有 class .form-group 的<div>中。这是获取最佳间距所必需的。

向所有的文本元素<input>、<textarea>和<select>添加 class .form-control。

2、内联表单：

请向<form>标签添加 class .form-inline。

使用class .sr-only，您可以隐藏内联表单的标签。

3、水平表单：

向父<form>元素添加 class .form-horizontal。

把标签和控件放在一个带有 class .form-group 的<div>中。

向标签添加 class .control-label。

按钮

.btn 为按钮添加基本样式

.btn-default 默认/标准按钮

.btn-primary 原始按钮样式（未被操作）

.btn-success 表示成功的动作

……

.active 按钮被点击

.disabled 禁用按钮

图片

.img-rounded 添加 border-radius:6px 来获得图片圆角。

.img-circle 添加 border-radius:500px 来让整个图片变成圆形。

.img-thumbnail 添加一些内边距（padding）和一个灰色的边框。

bootstrap 布局组件

字体图标

在span中定义.glyphicon glyphicon-*。具体见：字体图标列表

下拉菜单

需要在 class .dropdown 内加上下拉菜单即可。

导航元素

标签式——以一个带有class .nav的无序列表开始。添加 class .nav-tabs

胶囊式——class .nav-pills

禁用——对.nav class，添加.disabled class

下拉菜单的标签——添加带有.dropdown-menu class的无序列表

导航栏

步骤如下：

  • 向<nav>标签添加class .navbar、.navbar-default。

  • 向上面的元素添加role="navigation"，有助于增加可访问性。

  • 向<div>元素添加一个标题class .navbar-header，内部包含了带有class navbar-brand的<a>元素。这会让文本看起来更大一号。

  • 为了向导航栏添加链接，只需要简单地添加带有class .nav、.navbar-nav的无序列表即可。

固定到顶部——向.navbar class添加class .navbar-fixed-top

分页

class .pagination+无序列表

class .pager翻页

其它

徽章——class .badge

超大屏幕——Jumbotron：

  • 创建一个带有class .jumbotron的容器<div>。

  • 除了更大的<h1>，字体粗细font-weight被减为200px。

面板——向<div>元素添加class .panel和class .panel-default
