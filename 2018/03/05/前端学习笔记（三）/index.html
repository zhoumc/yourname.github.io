前端学习笔记（三）

发表于 2018-03-05 |

因为毕设的原因，要做系统，所以再补充一些前端知识的学习，这篇先复习补充一下HTML、CSS以及javascript基础。

另外介绍非常实用的前端编写工具Emmet。下篇会熟悉Bootstrap框架。

Emmet

非常好的包，使得Sublime Text写前端如此的轻松。参考见:Emmet文档

HTML补充内容

表格

表格的表头使用<th>标签进行定义。

列表

无序列表使用<ul>标签;有序列表始于<ol>标签;里面用<li>
自定义列表以<dl>标签开始。每个自定义列表项以<dt>开始。每个自定义列表项的定义以<dd>开始。

布局

用<div>或者<table>。colspan属性来可以横跨列。

表单

表单使用表单标签<form>来设置。

文本域通过<input type="text">标签来设定;密码字段通过标签<input type="password">来定义:

<input type="radio">标签定义了表单单选框选项。<input type="checkbox">定义了复选框

<input type="submit">定义了提交按钮。

CSS补充内容

背景

CSS 属性定义背景效果：

CSS中，颜色值通常以以下方式定义:

十六进制 - 如：”#ff0000”

RGB - 如：”rgb(255,0,0)”

颜色名称 - 如：”red”

文字

对齐方式——text-align,可取center、right、justify——每一行被展开为宽度相等，左，右外边距是对齐。

文本缩进——text-indent

文本间隔——word-spacing

字体

文本的大小——font-size。通常使用em单位代替像素。

链接

link、visited、hover、active四种状态。

表格

指定CSS表格边框，使用border属性。

填充

Padding属性定义元素边框与元素内容之间的空间。

显示

display属性和visibility属性都可以用来隐藏某个元素

定位

position，fixed代表相对于浏览器窗口是固定；Relative相对于浏览器窗口是固定

总结

这块如果自己建立模板的话会比较费事，然而很多框架的出现让其反而变得简单了。
所以这边比较快速的把所需的部分过了一遍，如果想更加巩固，做练习：CSS实例

javascript基础

简介

JavaScript 是一种轻量级的编程语言。

JavaScript 是可插入HTML页面的编程代码。

JavaScript 插入HTML页面后，可由所有的现代浏览器执行。

知识图谱

用法

HTML 中的脚本必须位于 标签之间。

输出

  • 使用 window.alert() 弹出警告框
  • 使用 document.write() 方法将内容写到HTML文档中。
  • 使用 innerHTML 写入到 HTML 元素。
  • 使用 console.log() 写入到浏览器的控制台。

语法

对象（Object）字面量 定义一个对象：

{firstName:”John”, lastName:”Doe”, age:50, eyeColor:”blue”}

函数（Function）字面量 定义一个函数：

function myFunction(a, b) { return a * b;}

定义变量——使用关键字 var；使用等号来为变量赋值

注释——单行双斜杠;多行注释以/*开始，以*/结尾。

JavaScript 对大小写是敏感的。

您可以在一条语句中声明很多变量。该语句以var开头，并使用逗号分隔变量即可,可横跨多行。

创建数组的三种方式：

var cars=new Array();

var cars=new Array(“Saab”,”Volvo”,”BMW”);

var cars=[“Saab”,”Volvo”,”BMW”];

函数用function+花括号定义，花括号是必须的。

可以使用内置属性length来计算字符串的长度:其它常用字符串方法

var txt = “ABCDEFGHIJKLMNOPQRSTUVWXYZ”;

var sln = txt.length;

href=”#”与href=”javascript:void(0)”的区别:

#包含了一个位置信息，默认的锚是#top 也就是网页的上端。

而javascript:void(0), 仅仅表示一个死链接。

在页面很长的时候会使用 # 来定位页面的具体位置，格式为：# + id。

如果你要定义一个死链接请使用 javascript:void(0)。

正则表达式

语法：/pattern/modifiers;

  • search()方法——用于检索字符串中指定的子字符串，或检索与正则表达式相匹配的子字符串，并返回子字符串的起始位置。

  • replace()方法——用于在字符串中用一些字符替换另一些字符，或替换一个与正则表达式匹配的子字符串。

修饰符可以在全局搜索中不区分大小写:
i——执行对大小写不敏感的匹配
g——执行全局匹配（查找所有匹配而非在找到第一个匹配后停止）
m——执行多行匹配

总结

javascript确实比较轻量级，可以说学过C、Java之后上手这个还是比较简单的，语法很类似。有了它可以使得系统前端更加的灵活。

通常用的比较多的还是那两个：
