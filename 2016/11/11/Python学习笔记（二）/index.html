Python基础语法

注释

单行注释——#可在表达式行末。

多行注释——三个单引号或者三个双引号。

导入其他模块

import 模块名（无后缀）。

缩进

每个层级四个空格。（没有{}，只是依据缩进）

注意：所有代码块语句必须包含相同的缩进空白数量。以下两个常见错误：

  • IndentationError: unexpected indent 错误——通常是tab和空格没对齐的问题。
  • IndentationError: unindent does not match any outer indentation level错误——使用的缩进方式不一致，有的是 tab 键缩进，有的是空格缩进，改为一致即可。

Python标识符

区分大小写。

以下划线开头的标识符是有特殊意义的。

  • 以单下划线开头（_foo）的代表不能直接访问的类属性，需通过类提供的接口进行访问，不能用”from xxx import *”而导入。
  • 以双下划线开头的（__foo）代表类的私有成员。
  • 以双下划线开头和结尾的（foo）代表python里特殊方法专用的标识，如init（）代表类的构造函数。

多行语句

语句中包含[], {} 或 ()就不需要使用多行连接符。

空行

分隔两段不同功能或含义的代码，便于日后代码的维护或重构。

空行也是程序代码的一部分。

标准数据类型

Numbers（数字）

不可改变的数据类型。改变数字数据类型会分配一个新的对象。

当指定一个值时，Number对象就会被创建。

使用del语句删除一些对象引用。示例：

Python支持四种不同的数值类型：

int（有符号整型）

long（长整型）Python使用”L”来显示长整型。

float（浮点型）

complex（复数）用a + bj,或者complex(a,b)表示， 复数的实部a和虚部b都是浮点型。

String（字符串）

python的字串列表有2种取值顺序:

从左到右索引默认0开始的，最大范围是字符串长度少1；从右到左索引默认-1开始的，最大范围是字符串开头

取一段子串——变量[头下标:尾下标]（其中下标是从0开始算起，可以是正数或负数，下标可以为空表示取到头或尾）

比如:

s[1:5]的结果是love。

当使用以冒号分隔的字符串，python返回一个新的对象，结果包含了以这对偏移标识的连续的内容，左边的开始是包含了下边界。而取到的最大范围不包括上边界。

List（列表）

列表用[ ]标识。是python最通用的复合数据类型。

列表支持字符，数字，字符串甚至可以包含列表（所谓嵌套）。

分割也可以用到变量[头下标:尾下标]（从左到右索引默认0开始的，从右到左索引默认-1开始，下标可以为空表示取到头或尾。）

Tuple（元组）

元组用()标识。

不允许更新，相当于只读列表。（而List允许更新）

Dictionary（元字典）

无序的对象集合。(而List是有序的对象集合）

是通过键来存取的。（不是通过偏移存取）

字典用{ }标识。由索引(key)和它对应的值(value)组成。
