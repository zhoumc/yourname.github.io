Python学习笔记（四）

发表于 2016-11-28 |

今天学习了Python数字和Python字符串，还算是比较简单，但具体使用的时候需要记忆一些方法以及参数（使用中记忆）。
以下是具体的学习内容。

Python数字

Python数字类型转换

int(x)——将x转换为一个整数。

str(x)——将对象x转换为字符串。

repr(x)——将对象x转换为表达式字符串。

eval(str)——用来计算在字符串中的有效Python表达式,并返回一个对象。

Python数学函数

ceil(x)——返回数字的上入整数，如math.ceil(4.1)返回5。
floor(x)与之相反。

cmp(x,y)——x < y返回-1,如果x == y返回0,如果x > y返回1。

max(x1,x2,…)——返回给定参数的最大值，参数可以为序列。
min(x1,x2,…)与之相反。

modf(x)——返回x的整数部分与小数部分，两部分的数值符号与x相同，整数部分以浮点型表示。

pow(x,y)——x**y运算后的值。

round(x [,n])——返回浮点数x的四舍五入值，如给出n值，则代表舍入到小数点后的位数。

Python随机数函数

Python三角函数

hypot(x,y)——返回欧几里德范数sqrt(x*x + y*y)。

\r——回车
